# Java-Audio_Transcribe

Audio Transcription Application

This is a Spring Boot–based Audio Transcription Application that provides REST APIs to upload audio files and receive transcribed text.

Features

    Upload audio files
    
    Process audio for transcription
    
    RESTful API support
    
    CORS configuration enabled

Technologies Used

    Java
    
    Spring Boot
    
    REST API

Project Files

    AudioTranscribeApplication.java – Main application entry point
    
    TranscriptionController.java – Handles transcription APIs
    
    WebConfig.java – CORS and web configuration

How to Run

    Open the project in an IDE (IntelliJ / Eclipse).
    
    Run AudioTranscribeApplication.java.
    
    Access APIs using Postman or browser.

Use Case
    
    Useful for speech-to-text, audio processing, and learning REST API integration.
